# Low-Level-Design- Java Practice Projects

This repository contains small Java projects focused on object-oriented design, basic problem solving, and console-based game flows.

## Projects

- `Chess`: Console-based chess model and controller implementation.
- `MortgageCalculator`: Interactive mortgage payment calculator.
- `SnakesAndLadder`: Console-based snakes and ladders gameplay.
- `TicTacToe`: Console-based tic-tac-toe with strategy support.

## Prerequisites

- Java JDK 8+ (or any modern JDK)

## Build And Run

Each project is self-contained. From a project directory:

```bash
javac -d out $(find src -name "*.java")
java -cp out Main
```

## Notes

- Most projects are interactive and expect console input.
- IntelliJ `.iml` files are included for IDE-based execution.
